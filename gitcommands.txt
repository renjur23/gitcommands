> git init [to initialize git]

> git status [to check all untracked files]

> git add -A [to adding all untracked file to staging area ]

> git commit -m "commit message" [to commit added files to local repositary]

> git clone [to take extact copy from remote repositary to local repositary]

> git pull [ to take changes from remote repositary to local repositary]

> git push [ to sent changes from local repositary to remote repositary]

> git config --list [to list cconfiguration]

> git config --global user.name "username" [to add username]

> git config --global user.email "useremail"[to add useremail]
